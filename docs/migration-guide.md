## Version 1.0.0 (Initial Release)

This is the initial release of the FundraiseUp .NET Client Library. No migration is required.

### Key Features Introduced
- Complete FundraiseUp API integration
- Fluent API design with builder patterns
- HttpClientFactory integration
- Advanced rate limiting with multiple strategies
- Comprehensive error handling
- Full dependency injection support

### Breaking Changes Policy

We follow [Semantic Versioning](https://semver.org/) principles:

- **MAJOR** version changes may include breaking changes
- **MINOR** version changes add functionality without breaking existing code
- **PATCH** version changes include bug fixes and minor improvements

## Migration Support

If you need help migrating between versions:

3. Consult the [API Reference](api-reference.md) for updated method signatures
